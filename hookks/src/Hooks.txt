1. It Allows you to use state and other React features without writing a class. Hooks are the functions which
"hook into" React State and lifecycle features from function components.

2. It does not works inside the class.

3. Hooks should always be use at the top level of the React functions.

4. Node version 6 or above. and NPM 5.2 version or above.

EXAPLE:
import { useState } from "react";

const App = () => {
  const state = useState();//hooks start
  console.log(state);
  const [count, setCount] = useState(0);
  const IncNum = () => {
    setCount(count + 1);
  }//Hooks end
  return (
    <>
      <h1>{count}</h1>
      <button onClick={IncNum}>Click Me!</button>
    </>
  )
}
export default App;
